The user wants to build a comprehensive mobile application platform called CLUBIN INDIA, inspired by TicketWings.in. The project involves creating two separate apps (Customer and Partner) and a web-based Admin Panel, all connected to a centralized backend.

**PRODUCT REQUIREMENTS (Based on all user messages):**

**Overall:**
-   **Platform:** Customer App, Partner App, Admin Panel.
-   **Theme:** Premium Black & Gold, based on the provided logo.
-   **Branding:** Use the provided logo for all app icons, splash screens, and in-app branding.
-   **Authentication:** OTP-based login using mobile number. Use UltraMsg for sending OTPs via WhatsApp.

**1. Customer App:**
-   **Onboarding:**
    -   Mandatory registration with Name, Mobile (OTP verified).
    -   Mandatory 21+ age verification with Government ID upload.
    -   Required T&amp;C checkbox with a warning about legal action for fake IDs.
-   **Home Screen:**
    -   Top banner/slider for featured clubs and events.
    -   Location system: Auto-detect and manual city selection.
    -   Sections for nearby events, popular clubs, with time-based filters (Today, This Week).
-   **Event &amp; Club Discovery:**
    -   Detailed pages for events including: flyer, layout, date/time, artist, video, organizers, sponsors, and ticket pricing.
-   **Booking System:**
    -   Select multiple ticket categories (GA, VIP, Table).
    -   Add guest details for multiple tickets.
    -   Integrate Razorpay for payments.
-   **QR Ticket System:**
    -   Generate a unique, secure, **Golden** QR code after payment.
    -   Display booking details (Name, Date, Time, Booking ID) in white text alongside the QR code.
-   **Notifications:**
    -   Instant booking confirmation via Email and **WhatsApp (UltraMsg)**, including the QR code.
    -   Push notifications for new events and offers.
-   **Other Features:**
    -   Favorites list for clubs/events.
    -   Booking history.

**2. Partner App (Future Work):**
-   Event and Ticket Management.
-   Real-time booking dashboard and analytics.
-   QR code scanning system for entry verification.
-   Marketing and promotion tools.

**3. Admin Panel (Future Work):**
-   User and partner management.
-   Content approval (clubs, events).
-   Fraud monitoring (fake IDs).
-   Platform-wide revenue and commission tracking.
