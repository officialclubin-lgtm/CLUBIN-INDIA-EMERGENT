# CLUBIN INDIA - Vercel Deployment Guide

## 🚀 Deployment Overview

This guide covers deploying CLUBIN INDIA to Vercel with both backend (FastAPI) and frontend (Expo Web).

---

## 📋 Prerequisites

1. **Vercel Account**: Sign up at https://vercel.com
2. **Vercel CLI**: Install globally
   ```bash
   npm install -g vercel
   ```
3. **MongoDB Atlas**: Cloud MongoDB database (free tier available)
4. **API Keys**: UltraMsg, Razorpay credentials

---

## 🗄️ Step 1: Setup MongoDB Atlas

1. Go to https://www.mongodb.com/cloud/atlas
2. Create a free cluster
3. Create database user with password
4. Whitelist all IP addresses (0.0.0.0/0) for Vercel
5. Get connection string:
   ```
   mongodb+srv://username:password@cluster.mongodb.net/clubin_india
   ```

---

## 🔧 Step 2: Configure Environment Variables

### On Vercel Dashboard:

1. Go to your project settings
2. Navigate to **Environment Variables**
3. Add the following:

```env
# MongoDB
MONGO_URL=mongodb+srv://username:password@cluster.mongodb.net/
DB_NAME=clubin_india

# UltraMsg WhatsApp API
ULTRAMSG_INSTANCE_ID=your_instance_id
ULTRAMSG_TOKEN=your_token

# Razorpay Payment Gateway
RAZORPAY_KEY_ID=your_key_id
RAZORPAY_KEY_SECRET=your_key_secret

# Frontend API URL (auto-generated by Vercel)
EXPO_PUBLIC_BACKEND_URL=https://your-app.vercel.app
```

## 📤 Step 6: Deploy to Vercel

### Option A: Using Vercel CLI

1. **Login to Vercel**:
   ```bash
   vercel login
   ```

2. **Deploy from project root**:
   ```bash
   cd /app
   vercel
   ```

3. **Follow prompts**:
   - Link to existing project or create new
   - Confirm settings
   - Deploy!

4. **Set environment variables**:
   ```bash
   vercel env add MONGO_URL
   vercel env add ULTRAMSG_INSTANCE_ID
   vercel env add ULTRAMSG_TOKEN
   vercel env add RAZORPAY_KEY_ID
   vercel env add RAZORPAY_KEY_SECRET
   ```

5. **Deploy production**:
   ```bash
   vercel --prod
   ```

### Option B: Using GitHub Integration

1. **Push to GitHub**:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/yourusername/clubinindia.git
   git push -u origin main
   ```

2. **Import on Vercel**:
   - Go to https://vercel.com/new
   - Import your GitHub repository
   - Configure environment variables
   - Deploy automatically

## 🔄 Step 7: Configure CORS for Production

Update `backend/server.py` CORS settings:

```python
app.add_middleware(
    CORSMiddleware,
    allow_credentials=True,
    allow_origins=[
        "https://your-app.vercel.app",
        "http://localhost:3000",  # For local development
    ],
    allow_methods=["*"],
    allow_headers=["*"],
)
```

## 🐛 Troubleshooting

### Issue: API Routes Not Working
**Solution**: Check `vercel.json` routes configuration

### Issue: MongoDB Connection Failed
**Solution**: 
- Verify connection string
- Check IP whitelist (0.0.0.0/0)
- Ensure database user has correct permissions

### Issue: Environment Variables Not Loading
**Solution**:
- Redeploy after adding env vars
- Check variable names match exactly
- Use `vercel env pull` to verify

### Issue: CORS Errors
**Solution**:
- Update allowed origins in CORS middleware
- Include Vercel deployment URL

### Issue: Build Fails
**Solution**:
- Check `requirements.txt` for Python
- Check `package.json` for Node
- Review build logs: `vercel logs`
